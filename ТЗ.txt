Разработка и тестирование мобильного приложения для управления задачами
1. Описание задачи: Разработать приложение для мобильных платформ (iOS, Android, Windows Phone), которое будет предоставлять возможность пользователям создавать и редактировать задачи, устанавливать сроки выполнения, отслеживать прогресс и получать уведомления о приближающихся сроках (каждый создаёт такое приложение, которое связано с вашим предприятием, где проходите практику)
2. Требования к приложению:
- Приложение должно быть разработано на основе современных технологий и подходов к разработке мобильных приложений.
- Должны быть реализованы следующие функции:
    * Создание задач.
    * Редактирование задач (изменение названия, описания, сроков выполнения).
    * Установка сроков выполнения задач.
    * Отслеживание прогресса выполнения задач (процент выполнения).
    * Получение уведомлений о приближающихся сроках выполнения задач.
    - Интерфейс приложения должен быть интуитивно понятным и удобным для пользователей.
3. Дополнительные требования:
- Реализовать механизм проверки корректности вводимых данных.
- Обеспечить возможность сохранения истории изменений задач.
- Предоставить возможность экспорта данных в формате CSV.
4. Тестирование:
- Провести функциональное тестирование всех функций приложения.
- Проверить работу приложения при различных сценариях использования.
5. Рефакторинг и оптимизация:
- Проанализировать код приложения на предмет возможных улучшений.
- Оптимизировать код для повышения производительности и удобства сопровождения.
6. Документация:
- Подготовить документацию по приложению, включая описание функций, примеры использования и инструкции по установке и настройке.
7. Разработка модулей для мобильных платформ:
- Разработать модули для iOS, Android или Windows Phone, которые будут обеспечивать взаимодействие с операционной системой и другими компонентами приложения.
8. Интеграция с API:
- Интегрировать приложение с API для получения информации о задачах и уведомлениях.
9. Безопасность:
- Обеспечить безопасность данных пользователей и защиту от несанкционированного доступа.
